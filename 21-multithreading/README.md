# Multithreading (Thread Kullanımı)

Bu klasörde Java'da **multithreading** kullanımı
Runnable arayüzü üzerinden basit bir kronometre örneğiyle gösterilmiştir.

## İçerik
- Runnable interface implementasyonu
- Thread oluşturma ve başlatma
- Aynı anda birden fazla thread çalıştırma
- Thread.sleep ile zamanlama
- InterruptedException yönetimi

## Amaç
Java'da thread mantığını,
aynı anda birden fazla işlemin
nasıl paralel çalıştırılabildiğini kavramak.
