# Abstract Classes

Bu klasörde Java'da **abstract class** kullanımı
veritabanı yöneticileri örneği üzerinden gösterilmiştir.

## İçerik
- Abstract sınıf tanımı
- Abstract metot kullanımı
- Alt sınıflarda metot implementasyonu
- Üst sınıf referansı ile farklı sınıfların kullanılması

## Amaç
Abstract sınıfların kullanım mantığını ve
ortak bir yapının alt sınıflar tarafından
nasıl uygulanacağını kavramak.
