# Packages (Paketler)

Bu klasörde Java'da **package** kullanımı
basit bir matematik örneği üzerinden gösterilmiştir.

## İçerik
- Package (paket) tanımı
- Farklı paketteki sınıfın import edilmesi
- Paket içindeki sınıfın kullanımı

## Amaç
Kodları paketler altında düzenleyerek
daha okunabilir ve yönetilebilir bir yapı oluşturmayı kavramak.
