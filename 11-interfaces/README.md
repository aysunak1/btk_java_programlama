# Interfaces

Bu klasörde Java'da **interface** kullanımı
veri erişim katmanı (DAL) örneği üzerinden gösterilmiştir.

## İçerik
- Interface tanımı (ICustomerDal)
- Interface implement eden sınıflar
- Farklı implementasyonların aynı arayüz üzerinden kullanılması
- Manager sınıfında interface referansı kullanımı

## Amaç
Interface mantığını ve
bağımlılıkları soyutlayarak
esnek ve genişletilebilir yapı kurmayı kavramak.
