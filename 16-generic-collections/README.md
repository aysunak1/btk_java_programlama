# Generic Collections

Bu klasörde Java'da **generic collection** kullanımı
müşteri listesi örneği üzerinden gösterilmiştir.

## İçerik
- Generic ArrayList kullanımı (ArrayList<Customer>)
- Parametreli constructor ile nesne oluşturma
- for-each döngüsü ile koleksiyon üzerinde gezinme

## Amaç
Generic collection yapısını kullanarak
tip güvenli (type-safe) ve düzenli veri yönetimini kavramak.
