# Kalıtım (Inheritance)

Bu klasörde Java'da **kalıtım (inheritance)** kullanımı örneklenmiştir.

## İçerik
- Üst sınıf (Person) tanımı
- Alt sınıflar (Customer, Employee) oluşturma
- Manager sınıflarında kalıtım kullanımı (PersonManager -> CustomerManager / EmployeeManager)
- Alt sınıflara özel metotlar ekleme

## Amaç
Kalıtım mantığını, üst sınıf–alt sınıf ilişkisini ve
kod tekrarını azaltma yaklaşımını kavramak.
