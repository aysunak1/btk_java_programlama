# JDBC - MySQL

Bu klasörde Java'da **JDBC** ile **MySQL** veritabanına bağlanma ve
temel CRUD işlemleri örneklenmiştir.

## İçerik
- DriverManager ile bağlantı kurma
- Connection / Statement / PreparedStatement kullanımı
- SELECT, INSERT, UPDATE, DELETE örnekleri
- ResultSet ile veri okuma
- SQLException hata yönetimi (DbHelper)

## Amaç
JDBC mantığını, veritabanı bağlantısı kurmayı ve
CRUD işlemlerini Java üzerinden yapmayı kavramak.
