# Java Fundamentals

Bu repository, Java programlama dilinin temel konularını
örnekler üzerinden öğrenmek ve pekiştirmek amacıyla oluşturulmuştur.

Çalışmalar; temel Java konularından başlayarak
nesneye yönelik programlama (OOP), SOLID prensipleri,
hata yönetimi, dosya işlemleri, koleksiyonlar,
çoklu iş parçacığı (multithreading) ve
veritabanı bağlantılarına kadar ilerlemektedir.

## İçerik
- Temel Java yapıları (döngüler, diziler, metotlar)
- Sınıflar ve nesneler
- Kalıtım, polymorphism, abstract class, interface
- SOLID prensiplerine giriş
- Exception handling ve custom exception
- Collections ve generics
- Dosya okuma / yazma işlemleri
- Multithreading
- JDBC ile MySQL bağlantısı

Her konu, kendi klasörü altında
örnek kodlar ve açıklayıcı README dosyaları ile sunulmuştur.

## Amaç
Java programlama dilinin temellerini
düzenli, anlaşılır ve sürdürülebilir bir yapı içerisinde
öğrenmek ve GitHub üzerinde referans alınabilecek
bir kaynak oluşturmak.
