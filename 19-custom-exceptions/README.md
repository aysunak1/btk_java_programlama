# Custom Exceptions (Özel Hatalar)

Bu klasörde Java'da **custom exception** tanımlama ve kullanımı
banka hesabı örneği üzerinden gösterilmiştir.

## İçerik
- Exception sınıfından türetilmiş özel hata tanımı
- throws ile exception fırlatma
- try-catch ile custom exception yakalama
- Hata mesajının dışarıya iletilmesi

## Amaç
Uygulamaya özel hata durumlarını
kendi exception sınıflarımızla yönetmeyi
ve daha anlamlı hata mesajları üretmeyi kavramak.
